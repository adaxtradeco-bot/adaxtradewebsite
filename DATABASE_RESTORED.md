## 🚀 دستورات مفید

### بازیابی کامل دیتابیس:
```bash
# 1. Seed اصلی
npm run db:seed

# 2. همه صفحات
node seed-all-pages.js

# 3. صفحات باقیمانده
node seed-remaining-pages.js
```

### بررسی دیتابیس:
```bash
# تست اتصال
node test-db-connection.js

# لیست صفحات
node check-all-pages.js

# Prisma Studio
npm run db:studio
```
