## 📋 نکات مهم

1. **تغییر رمز عبور:**
   - بعد از اولین ورود، رمز عبور را تغییر دهید
   - از یک رمز عبور منحصر به فرد استفاده کنید

2. **امنیت:**
   - این فایل را در جای امن نگه دارید
   - این اطلاعات را با کسی به اشتراک نگذارید
   - از HTTPS استفاده کنید

3. **Backup:**
   - این اطلاعات را در جای امن ذخیره کنید
   - اگر رمز عبور را فراموش کردید، دیتابیس را reset کنید

## 🔄 تغییر رمز عبور

اگر رمز عبور را فراموش کردید:

```bash
# در سرور
cd ~/apps/english-website
npm run db:seed
```

سپس دوباره seed اجرا می‌شود و رمز عبور به مقدار پیش‌فرض بازگردانده می‌شود.
