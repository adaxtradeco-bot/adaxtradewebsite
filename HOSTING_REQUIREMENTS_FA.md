# نیازمندی‌های هاست برای اپلیکیشن Next.js

## 📋 معرفی پروژه

**نام پروژه:** پلتفرم آموزش زبان IvaFlow  
**تکنولوژی:** Next.js 14 (فریمورک React)  
**نوع:** اپلیکیشن وب Full-Stack با SSR (رندر سمت سرور)  
**دامنه:** ivaflow.com

---

## 🔧 نیازمندی‌های فنی

### 1. **محیط Node.js**
- **نسخه Node.js:** 18.x یا بالاتر (LTS توصیه میشود)
- **نسخه npm:** 9.x یا بالاتر
- **الزامی:** پشتیبانی کامل از Node.js runtime (نه هاست استاتیک)

### 2. **مشخصات سرور**
- **RAM:** حداقل 1GB (2GB توصیه میشود)
- **فضای ذخیره‌سازی:** حداقل 2GB فضای خالی
- **CPU:** هاست اشتراکی قابل قبول، VPS ترجیحاً
- **سیستم عامل:** Linux (Ubuntu/CentOS) یا سازگار

### 3. **پورت و شبکه**
- **دسترسی به پورت:** پورت 3000 یا تنظیم پورت سفارشی
- **Reverse Proxy:** Nginx یا Apache با پشتیبانی proxy
- **SSL/TLS:** گواهی HTTPS الزامی
- **WebSocket:** پشتیبانی برای امکانات real-time

### 4. **دیتابیس (در صورت نیاز)**
- **PostgreSQL:** نسخه 14+ یا
- **MySQL:** نسخه 8+ یا
- **MongoDB:** نسخه 5+
- **اتصال:** پشتیبانی از اتصال دیتابیس remote

### 5. **متغیرهای محیطی**
- پشتیبانی از فایل‌های `.env` یا تنظیم environment variables
- ذخیره‌سازی امن برای اطلاعات حساس

## 📦 ساختار اپلیکیشن

```
english-website/
├── .next/              # اپلیکیشن build شده (تولید میشود)
├── public/             # فایل‌های استاتیک
├── src/                # کد منبع
├── node_modules/       # وابستگی‌ها (بیش از 10,000 فایل)
├── package.json        # لیست وابستگی‌ها
├── next.config.js      # تنظیمات Next.js
└── server.js           # سرور سفارشی (در صورت نیاز)
```

**حجم کل:** ~300-500MB (با node_modules)

## 🌐 تنظیمات دامنه

### تنظیمات DNS:
```
Type: A Record
Host: @ (یا www)
Points to: [IP سرور]
TTL: 3600
```

### Reverse Proxy (مثال Nginx):
```nginx
server {
    listen 80;
    server_name ivaflow.com www.ivaflow.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

## 📞 سوالاتی که باید از هاست بپرسید

1. **"آیا از اپلیکیشن‌های Node.js پشتیبانی میکنید؟"**
   - اگر بله، کدام نسخه‌ها؟

2. **"آیا میتوانم اپلیکیشن Next.js را روی سرورهای شما اجرا کنم؟"**
   - Next.js نیاز به پروسه دائمی Node.js دارد

3. **"آیا cPanel با Node.js Selector ارائه میدهید؟"**
   - استقرار را خیلی راحت‌تر میکند

4. **"آیا دسترسی SSH موجود است؟"**
   - برای استقرار دستی لازم است

5. **"آیا از پیکربندی reverse proxy پشتیبانی میکنید؟"**
   - برای مسیریابی دامنه به اپلیکیشن Node.js نیاز است

6. **"چه process manager را توصیه میکنید؟"**
   - PM2، Forever یا راه‌حل داخلی؟

7. **"آیا گواهی SSL خودکار ارائه میدهید؟"**
   - یکپارچگی با Let's Encrypt؟

8. **"حداکثر محدودیت حافظه برای اپلیکیشن‌های Node.js چقدر است؟"**
   - باید حداقل 1GB باشد
